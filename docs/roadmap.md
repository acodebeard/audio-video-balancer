# Roadmap

## MVP

1. Prove the active tab audio delay pipeline.
2. Build the extension popup from the approved mock.
3. Add a controlled local sync fixture.
4. Package installable Chrome ZIPs from source.
5. Document installation, privacy, and limitations.

## Later

- Firefox support if the Chrome approach proves stable.
- Per-site delay presets.
- Keyboard shortcuts.
- Video delay support for the opposite sync problem.
